# Design Sprint Output Template

<!-- VERSION: 1.0.0 | SKILL: /ux-design-sprint | AGENT: ux-sprint-facilitator -->

> Template for structuring Design Sprint 2.0 (AJ&Smart, Courtney 2019) output artifacts.
> Populate each section during the corresponding sprint day.

## Document Sections

| Section | Purpose |
|---------|---------|
| [Sprint Metadata](#sprint-metadata) | Engagement context and sprint configuration |
| [Day 1: Map](#day-1-map) | Challenge map, long-term goal, sprint questions, HMW notes, target selection |
| [Day 2: Sketch](#day-2-sketch) | Lightning demos, four-step sketches, solution sketches |
| [Day 3: Decide](#day-3-decide) | Art Museum voting, straw poll, supervote, storyboard |
| [Day 4: Test](#day-4-test) | Prototype specification, 5-user interview observations, pattern analysis |
| [Sprint Verdict](#sprint-verdict) | Sprint question verdicts and next steps |
| [Synthesis Judgments](#synthesis-judgments) | AI judgment calls with confidence classification |
