## Summary

Write Diataxis explanation documents for the top 5 Jerry skills. PROJ-015 audit found 87% explanation gap — users know *what* Jerry skills do (reference docs exist) but not *why* they exist or how they fit together. Explanations provide conceptual understanding through discursive prose.

**Key Objectives:**
- Write /problem-solving explanation (why structured problem-solving matters)
- Write /orchestration explanation (why multi-agent coordination)
- Write /worktracker explanation (why persistent work tracking)
- Write /eng-team explanation (why security-first engineering)
- Write /prompt-engineering explanation (why structured prompts)

---

## Business Outcome Hypothesis

**We believe that** providing explanation documents for the top 5 skills

**Will result in** users understanding the *why* behind Jerry's design decisions and making better skill selection choices

**We will know we have succeeded when** each explanation passes Diataxis explanation criteria (understanding-oriented, discursive, contextual, multiple perspectives)
