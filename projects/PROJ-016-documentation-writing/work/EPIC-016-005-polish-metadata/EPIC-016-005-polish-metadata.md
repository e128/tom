Fix minor documentation issues and add structural metadata. This epic covers getting-started.md tutorial branching issues, title framing corrections, Diataxis quadrant metadata on all docs, and tutorials for the remaining 10 skills not covered by EPIC-016-002.

**Key Objectives:**
