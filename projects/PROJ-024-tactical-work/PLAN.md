# PROJ-024-tactical-work — Plan

> Tactical work items — small improvements, quick fixes, and miscellaneous tasks that don't belong to a larger project initiative.

## Objective

Provide a catch-all project for tactical work: small improvements, one-off fixes, housekeeping tasks, and miscellaneous items that are too small or cross-cutting for a dedicated project but still need tracking.

## Scope

- Small codebase improvements and cleanup
- One-off configuration or tooling fixes
- Cross-cutting housekeeping tasks
- Quick enhancements that don't warrant a full project
- Miscellaneous items that need worktracker visibility

## Out of Scope

- Bug fixes (use PROJ-030-bugs)
- Feature enhancements with clear project alignment (use the relevant PROJ-NNN)
- Multi-sprint initiatives (create a dedicated project)

## Status

**Phase:** Active
